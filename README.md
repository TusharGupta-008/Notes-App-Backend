# Notes App

A full-stack Notes Management Application built using the MERN stack.

## Features

- User Signup
- User Login
- JWT Authentication
- Protected Routes
- Create Notes
- View Notes
- Delete Notes

## Tech Stack

### Frontend

- React.js
- React Router DOM
- Axios
- CSS

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcryptjs
